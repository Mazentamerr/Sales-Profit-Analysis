## 🚀 Overview

This Power BI project provides a comprehensive and interactive dashboard
for analyzing sales and profit performance. It offers actionable
insights into revenue, growth, orders, and product performance over time
and across various dimensions (Year, Month, Quarter, Country, City,
Region).
